jQuery Vibrate example

Controlled Vibration

Click the button to vibrate it.

Continuous (original behavior)

This element
vibrates on its own every now and then. (Interval defined by 'frequency')

Setup

// Setup with default options
$(element).vibrate({
    speed:		30,
	duration:	2000,
	frequency:	5000,
	spread:		3,
	continuous: false
});
